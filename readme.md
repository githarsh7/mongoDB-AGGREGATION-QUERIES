# MongoDB Aggregation Queries

A collection of MongoDB aggregation queries that demonstrates commonly used aggregation pipeline stages with practical examples. This repository is intended as a learning resource and quick reference for developers who want to understand and practice MongoDB aggregation.

## Overview

MongoDB's Aggregation Framework allows you to process and transform documents through a sequence of pipeline stages. This repository contains example queries that cover the most frequently used aggregation operations.

## Topics Covered

- `$match`
- `$project`
- `$group`
- `$sort`
- `$limit`
- `$skip`
- `$unwind`
- `$lookup`
- `$count`
- `$facet`
- `$addFields`
- `$set`
- `$unset`
- `$replaceRoot`
- `$replaceWith`
- `$bucket`
- `$bucketAuto`
- `$sortByCount`
- `$sample`

## Prerequisites

- MongoDB installed locally or access to MongoDB Atlas
- Basic knowledge of MongoDB CRUD operations
- MongoDB Shell (`mongosh`) or MongoDB Compass

## How to Use

1. Clone the repository.

```bash
git clone https://github.com/githarsh7/mongoDB-AGGREGATION-QUERIES.git
```

2. Navigate to the project directory.

```bash
cd mongoDB-AGGREGATION-QUERIES
```

3. Open the query files and execute them using:
   - MongoDB Shell (`mongosh`)
   - MongoDB Compass
   - MongoDB Atlas

4. Modify the sample queries to work with your own datasets.

## Learning Goals

This repository helps you learn how to:

- Filter documents
- Transform document fields
- Group and summarize data
- Join collections using `$lookup`
- Work with arrays using `$unwind`
- Build multi-stage aggregation pipelines
- Analyze and process data efficiently
